# FITApp React Web App
This is the frontend application for the FITApp.

## Running locally
### Prerequisites
- Yarn

### How to run
1. Clone the repository
    ```
    git clone https://github.com/Cubball/FITApp
    ```
1. cd into this folder
    ```
    cd FITApp/frontend
    ```
1. Install the dependencies
    ```
    yarn install
    ```
1. Run the app
    ```
    yarn run dev
    ```
